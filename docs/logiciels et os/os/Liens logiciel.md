Ou trouver les bons logiciels :

- Liste de logiciels libres reconnus par l'[administration française](https://code.gouv.fr/sill/list).

Les must :

- [Adguard](https://adguard.com/fr/welcome.html) bloquer de pub et (licence achetée).
- [Crytpomator](https://cryptomator.org/) logiciel pour crypter les données notamment sur le cloud (licence achetée).
* [Mixxx](https://mixxx.org/), logiciel de dj.
- [Obsidian](https://obsidian.md/) application de gestion de connaissances.
- [Thunderbird](https://www.thunderbird.net/fr/) gestionnaire d'emails.
-  [VSCodium](https://vscodium.com/) version du logiciel de programmation Visual Studio Code sans les trackers Microsoft.
* [VLC](https://images.videolan.org/vlc/index.fr.html), lecteur média.

Utile :
- [Dropbox](https://www.dropbox.com/install-linux)
* [Simplenote](https://simplenote.com/), logiciels de prises de notes en Markdown avec synch²ronisation entre les plateformes (Android, Windows,...). Pas si utile que ça.
* [Logseq](https://logseq.com/) application de gestion de connaissances de type Obsidian.
* [OnlyOffice](https://www.onlyoffice.com) et [FreeOffice](https://www.freeoffice.com/fr/) suite bureautique.

Jamais testé :

- [LM Studio](https://lmstudio.ai/download) permet d'installer des LLM en local.
